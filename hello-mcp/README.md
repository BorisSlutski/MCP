# hello-mcp

A minimal MCP server with resources, resource templates, prompts, and tools. Built step-by-step following the [Building MCP Servers](https://medium.com/@cstroliadavis/building-mcp-servers-536969d27809) tutorial series.

## Prerequisites

- Node.js and npm

## Commands

```bash
npm install
npm run build
```

## Run with MCP Inspector

```bash
npx @modelcontextprotocol/inspector node build/index.js
```

Then click **Connect** and use the Resources, Resource Templates, Prompts, and Tools tabs to test the server.

## Full tutorial

For the complete 4-part walkthrough (setup, resources, templates, prompts, tools), see [../building-mcp-servers-tutorial/README.md](../building-mcp-servers-tutorial/README.md).
